**Новая структура**:
```
┌──────────┬────────────────────────┐
│          │                        │
│ 🛡️ Admin │  Статистика (6 карт)  │
│          │                        │
│ 👥 Поль- │  ┌──────────────────┐  │
│   зовате-│  │ Таблица          │  │
│   ли     │  │ пользователей    │  │

│ 📝 Отчё- │                        │
│   ты     │                        │
│          │                        │
└──────────┴────────────────────────┘
```

#### Боковое меню (Sidebar):
- **Ширина**: 250px
- **Фон**: Тёмно-зелёный (secondary-bg)
- **Позиция**: Fixed, слева
- **Высота**: 100vh (на весь экран)

#### Элементы меню:
1. **Заголовок**:
   - "🛡️ Admin"
   - "Challenge Stake" (подзаголовок)

2. **Пункты меню**:
   - 👥 Пользователи
   - 📝 Отчёты

#### Стили пунктов меню:
- Неактивный: Прозрачный фон, серый текст
- Hover: Светло-зелёный фон (10% opacity)
- Активный: Зелёный фон (20% opacity), зелёный текст, жирный шрифт

#### Иконки:
- SVG формат
- Размер: 20x20px
- Stroke width: 2px
- Цвет меняется вместе с текстом

#### Основной контент:
- **Отступ слева**: 250px (ширина sidebar)
- **Padding**: 20px
- **Flex**: 1 (занимает всё оставшееся место)

### Технические детали

#### HTML структура:
```html
<body>
  <div class="sidebar">
    <div class="sidebar-header">...</div>
    <div class="sidebar-menu">
      <div class="sidebar-item active">...</div>
      <div class="sidebar-item">...</div>
    </div>
  </div>
  
  <div class="main-content">
    <div class="container">
      <!-- Статистика и контент -->
    </div>
  </div>
</body>
```

#### JavaScript:
```javascript
function switchTab(tab) {
  // Обновляем активный пункт меню
  document.querySelectorAll('.sidebar-item').forEach(item => 
    item.classList.remove('active')
  );
  event.target.closest('.sidebar-item').classList.add('active');
  
  // Переключаем контент
  document.querySelectorAll('.content').forEach(c => 
    c.classList.remove('active')
  );
  document.getElementById(`${tab}-content`).classList.add('active');
}
```

### Удалённые элементы:
- ❌ Старые табы сверху (`.tabs`, `.tab`)
- ❌ Заголовок "🛡️ Admin Panel" и подзаголовок из основного контента
- ❌ Колонка "Лайки" из таблицы отчётов

### Преимущества новой структуры:

1. **Навигация всегда видна** - не нужно скроллить вверх
2. **Больше места для контента** - вертикальное меню экономит место
3. **Профессиональный вид** - как в современных админ-панелях
4. **Легко расширять** - можно добавить новые разделы
5. **Таймер заметнее** - пользователи сразу видят дедлайн
