# Обновления системы голосования и таймеров

## 1. Модальное окно с причиной для голоса "Не верю"

### Что изменилось:
- При нажатии кнопки "Не верю" (thumbs down) теперь открывается модальное окно
- Пользователь должен выбрать причину из списка:
  - Фото из интернета
  - Старое фото
  - Не соответствует задаче
  - Подозрительный контент
  - Другое (с возможностью написать свою причину)

### Технические детали:
- Добавлено поле `reason` в таблицу `reportVotes` (опциональное)
- Функция `voteReport` теперь принимает параметр `reason`
- Создана функция `showFakeReasonModal()` для отображения модального окна
- Форма валидируется - нельзя отправить без выбора причины
- Если выбрано "Другое", появляется текстовое поле (макс. 200 символов)

### UX:
- Голос "Верю" работает как раньше - сразу без модального окна
- Голос "Не верю" требует указания причины
- После отправки показывается toast "Голос учтён"
- Можно отменить голос, нажав на ту же кнопку повторно

## 2. Таймер обратного отсчёта в карточках челленджей

### Что добавлено:
- В каждой карточке активного челленджа отображается оставшееся время
- Формат отображения адаптивный:
  - Больше 30 дней: "2 мес 5 дн"
  - От 1 до 30 дней: "10 дн 5 ч"
  - Меньше суток: "5 ч 30 мин"
  - Меньше часа: "45 мин"
  - Завершён: красный текст "Завершён"

### Технические детали:
- Создана функция `formatTimeRemaining(deadline)`
- Таймер отображается только для активных челленджей
- Цвет таймера: зелёный лайм (var(--accent-lime))
- Расположение: в метаданных карточки после статуса

### Где отображается:
- Лента (все челленджи)
- Профиль пользователя (вкладка "Челленджи")
- Мои челленджи

### Стили:
```css
.challenge-timer {
  color: var(--accent-lime);
  font-weight: 500;
  opacity: 1 !important;
}
```

## Примеры использования

### Голосование "Не верю":
1. Пользователь видит отчёт
2. Нажимает кнопку thumbs down
3. Открывается модальное окно "Почему не верите?"
4. Выбирает причину из списка
5. Нажимает "Отправить"
6. Голос учитывается, счётчик обновляется

### Таймер челленджа:
```
@username
15 янв • Активен • 5 дн 12 ч
```
